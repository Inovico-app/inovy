## 1. Purpose

This Statement of Applicability (SoA) is the master artifact of Inovy's ISMS. It documents the applicability of all 93 controls in ISO/IEC 27001:2022 Annex A, with justification for inclusion or exclusion, current implementation status, and references to implementation evidence and applicable policies.

---

## 2. Status Key

| Status          | Definition                                                                                        |
| --------------- | ------------------------------------------------------------------------------------------------- |
| **IMPLEMENTED** | Control is fully implemented; code and/or documented procedure exists and is operational          |
| **PARTIAL**     | Code or technical control exists but formal documentation, procedure, or full coverage is missing |
| **DOC NEEDED**  | No formal policy or procedure document exists; may have informal practices                        |
| **CODE NEEDED** | Code change required; no technical implementation in place                                        |
| **CODE+DOC**    | Both a code change and a new/updated policy/procedure document are required                       |
| **N/A**         | Control is not applicable to Inovy; formally excluded with justification                          |
